# 🎨 Whiteboard Web Application

A simple and interactive **online whiteboard** built using **HTML, CSS, and JavaScript (Canvas API)**.

This project allows users to draw freely, create shapes, choose colours, adjust brush size, erase content, clear the canvas, and save drawings as images.

---

## 🚀 Live Features

### 🖌️ Drawing Tools
- Freehand brush drawing
- Eraser tool (draws over with background colour)

---

### 🔺 Shape Tools
- Rectangle tool
- Circle tool
- Triangle tool

Shapes are dynamically drawn based on mouse movement.

---

### 🎨 Colour System
- Preset colour palette
- Custom colour selection (colour picker support)
- Real-time stroke colour updates

---

### 📏 Brush Controls
- Adjustable brush size slider
- Smooth stroke rendering using Canvas API

---

### 🧼 Canvas Controls
- Clear entire canvas with one click
- Save drawing as image (PNG/JPG download)

---

## 🛠️ Built With

- 🟧 HTML5  
- 🎨 CSS3  
- 🟨 JavaScript (Vanilla)  
- 🖼️ Canvas API
