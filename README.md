# Whiteboard-Web-Application
A simple interactive whiteboard built using **HTML, CSS, and JavaScript (Canvas API)**.   It allows users to draw freely, create shapes, change colours, adjust brush size, erase, clear the canvas, and save drawings as images.

---

## ✨ Features

- 🖌️ Freehand brush drawing
  
- 🧽 Eraser tool
  
- 🔺 Shapes:
  
  - Rectangle
    
  - Circle
    
  - Triangle
    
- 🎨 Colour picker + preset colour palette
  
- 📏 Adjustable brush size slider
  
- 🧼 Clear canvas button
  
- 💾 Save drawing as image (PNG/JPG)

---

- 🟨 JavaScript
